# 🔫 T2 Firearms Mass Production — Cheat Sheet (Level 45)

> Zone: Oak Forest (T2–3). At Lv.45 you have full access to Copper Mine, Tin Mine,
> and Oak Forest — the exact resources needed for T2 bronze firearms.

---

## ✅ PRE-FLIGHT CHECKLIST

| Requirement | How to Check |
|-------------|-------------|
| **Gun Workbench Lv.2** | Tap bench → Upgrade → T2 recipes must be unlocked |
| **Foundry Lv.2** | Required to smelt Bronze Ingots — upgrade this FIRST |
| **Pepperbox Blueprint** | Gun Workbench menu → Pistols section |
| **Bronze Weapon Parts** blueprint | Usually unlocked at Gun Workbench Lv.2 by default |
| **Bronze Plate** blueprint | Foundry menu — unlocked at Foundry Lv.2 |
| Wagon ready + 3–4 spare pickaxes | For bulk ore hauls |
| Storage chests cleared | Free up space before each run |

> ⚠️ Floor tiles under Gun Workbench AND Foundry must be upgraded to matching tier
> or the Upgrade button stays greyed out.

## 🔗 FULL MATERIAL CHAIN (Pepperbox — Best T2 Gun at Lv.45)

```
Pepperbox (T2 Pistol)
├── 2× Bronze Weapon Parts  ──┐
├── 1× Bronze Plate          ├── all made from Bronze Ingots
└── 1× Bronze Rivet          ──┘

1× Bronze Ingot  (Foundry Lv.2):
    = 4× Copper Ore
    + 4× Tin Ore
    + 2× Brown Coal
```

### Raw Ore Needed Per Gun

| Material | Per 1 Gun | Per 10 Guns | Farm Location |
|----------|-----------|-------------|---------------|
| **Copper Ore** | ~16–20 | ~180 | Copper Mine (T1 zone) |
| **Tin Ore** | ~16–20 | ~180 | Tin Mine (T2 zone) |
| **Brown Coal** | ~8–10 | ~90 | Any mine (spawns alongside ore) |

> 💡 Stockpile ALL raw ore first, then smelt in one overnight batch. Queuing
> in bulk is faster than crafting 1–2 guns per day.

## ⛏️ FARMING ROUTES (Energy-Optimized)

### Route A — Pure Ore Run (Fastest bronze stockpile)
```
Ranch
  → Copper Mine  (fill inventory: Copper Ore + Brown Coal)
  → Tin Mine     (fill remaining space: Tin Ore)
  → Ranch
  → Dump all into Foundry queue → sleep → smelt overnight
```

### Route B — Stacked XP Run (Farm + Progress)
```
Ranch
  → Oak Forest   (bandits for XP + quests + Oak timber)
  → Tin Mine     (grab Tin on return trip)
  → Silverton    (trade surplus furs/hides for Silver Coins)
  → Ranch
```

### Route C — Trade Shortcut (Low-energy or no-time day)
```
Silverton → Metals Merchant: 5× Copper → 1× Bronze Ingot equivalent
          → Native Tribe:    Wolf Teeth / Bear Claws → Tin Ore
Ranch     → smelt whatever you traded for → craft guns
           (skip mining entirely on busy days)
```

> 🔑 Inside mines, always use pickaxes you FIND on the ground first.
> Saves your crafted pickaxes' durability for later.

## 🏭 DAILY PRODUCTION LOOP

```
── LOGIN ──────────────────────────────────────────────────────────
1. Start Foundry: queue ALL Copper + Tin + Coal → Bronze Ingots

── WHILE FOUNDRY RUNS ─────────────────────────────────────────────
2. Use full energy on 1 ore run (Route A preferred)
3. Return → smelt new ore → add to queue

── BEFORE SLEEP ───────────────────────────────────────────────────
4. Craft: Bronze Weapon Parts, Bronze Plates, Bronze Rivets
5. Queue Gun Workbench with as many Pepperboxes as mats allow
6. Log out → everything crafts passively overnight

── NEXT LOGIN ─────────────────────────────────────────────────────
7. Collect finished guns → store or sell → repeat from step 1
```

### Overnight Queue Priority Order
```
1st → Foundry        (slowest, queue longest batches)
2nd → Components     (Weapon Parts, Plates, Rivets)
3rd → Gun Workbench  (fastest, queue last before sleep)
```

## 💰 SELL vs. KEEP DECISIONS

| Situation | Action |
|-----------|--------|
| More guns than you need | Trade at Silverton → Silver Coins |
| Guns at <30% durability | Use Repair Bench — cheaper than re-crafting |
| Season Pass active | Check "craft X weapons" task → cash in for Golden Bullets |
| In an Alliance | Donate surplus → Alliance Points |
| No blueprint yet | Farm Oak Forest bandit camps — drops T2 tubes regularly |

## ⚠️ TOP GOTCHAS

| Mistake | Consequence | Fix |
|---------|-------------|-----|
| Foundry below Lv.2 | Can't smelt Bronze — stuck | Upgrade Foundry BEFORE stockpiling ore |
| Not enough Brown Coal | Smelting stalls mid-batch | Carry 2× more coal than you estimate |
| Anger Meter maxes during mine run | Surprise raid when offline | Stop farming at ~75% Anger, head home |
| Inventory full on mine return | Wasted trip, drop mats | Use Wagon — park near mine, fill on return leg |
| Wrong blueprint tube used at Gustav | Wasted Uncommon tube | Only give COMMON tubes to Gustav |
| Levelling too fast | T3 enemies appear before T2 gear is ready | Don't rush XP — only do quests tied to farming |
