# Westland Survival — Expert Reference

> **Quick Summary**: Westland Survival is a western-themed mobile survival RPG by Helio Games
> where you build a ranch, craft gear, farm resources across tiered zones, fight bandits, tame
> pets, and compete in alliance events. Progression is gated by gear tiers (1–5+), zone access,
> and the new Skill Tree system introduced in v11.0. Success depends on energy efficiency,
> gear management, and smart resource prioritization.

---

## 1. Overview

### What Is Westland Survival?
A free-to-play mobile survival game (iOS/Android) set in the Wild West. You play as a lone
survivor who builds and defends a ranch, explores dangerous zones, fights bandits, tames
animals, crafts weapons & armor, and participates in seasonal events and alliance raids.

### Core Game Loop
```
Login → Collect Daily Rewards → Use Energy Farming/Questing
→ Return to Ranch → Craft/Upgrade → Defend from Bandit Raids
→ Participate in Events → Sleep / Repeat
```

### Why It Matters to Understand Deeply
- Energy is the primary limiting resource — wasting it = losing progress
- Gear tier gates zone access; entering wrong zones destroys equipment
- Skill tree choices (v11.0+) are semi-permanent — bad choices cost Silver to reset
- Alliance participation multiplies rewards significantly late game
- F2P players CAN remain competitive with smart Silver/resource management

## 3. Character Progression — The Skill Tree System (v11.0+)

> ⚠️ The old manual stat-point system is GONE as of December 2025. Stats are now upgraded
> via the Skill Tree and through equipment.

### Three Skill Branches

| Branch | Focus | Best For |
|--------|-------|----------|
| **Headhunter's Track** | Combat, exploration, battle bonuses | PvP/Raiding players, high-level zone farmers |
| **Wrangler's Track** | Pets, mounts, taming, fishing (Lv.70+) | Pet breeders, passive resource gatherers |
| **Craftsman's Track** | Crafting, gathering, trade perks | F2P players, base-builders, resource traders |

### How Tracks Work
1. You can run **up to 5 Tracks simultaneously**
2. Complete tasks within active Tracks → earn **Books**
3. Spend Books + Silver Coins to **unlock Skill Tree nodes**
4. Free Resets: each branch can be reset **3 times for free**; after that costs Silver

### Recommended Early Build
- **Start with Craftsman** → reduces crafting costs, boosts gathering
- **Then invest in Wrangler** → unlocks pet companion (saves weapon durability)
- **Headhunter last** → only becomes essential at Tier 4+ combat zones

### Weapon Type Bonuses (Post v11.0)
| Weapon Type | Bonus Against |
|-------------|---------------|
| Rifles / Spears | Animals (better loot, less durability loss) |
| Pistols / Knives | Bandits (faster kills, more XP) |
| Shotguns | Mixed encounters |

## 6. Ranch Management

### Build Priority Order (Beginner)
```
1. Tool Bench (T1) → upgrades ASAP
2. Storage Chests (more = carry more loot)
3. Gun Workbench + Armor Workshop
4. Cooking Station
5. Repair Bench
6. Defensive Walls + Stakes/Sandbags
7. Farming (Field, Well, Chicken Coop)
8. Pet House + Bait Workbench
9. Stable
10. Decorations (last)
```

### Building Mode Tips
- Tap the **white house icon** (top right, under minimap) to enter build mode
- Use the **workbench icon** (next to house icon) to remotely manage all benches
- Tap any structure → see exact upgrade costs before committing
- Always upgrade all benches to the **same tier** — bottlenecks kill progression

### Base Defense
- Upgrade walls to at least T2 before reaching Level 25
- Place **stakes and sandbags** between your walls and storage — slows bandits
- Be **online during a raid** — you can fight back and loot a Bandit's Stash Map from
  the raiding leader if you defeat them
- The **Anger Meter** fills as you kill bandits in the world. When full → raid incoming
  (you get a countdown warning before they arrive)

### Production Chain (Passive Farming)
```
Well → Water
Field (Corn) + Water → Corn Meal → Alcohol → Healing Items
Chicken Coop → Eggs → Cooking recipes
Field (Oats) → Horse/Wagon feed exclusively
```

## 7. Energy System — The #1 Mechanic

### Rules of Energy
- Max Energy = 100
- **Never let it sit at 100** — it's wasted regeneration
- Walking on the global map costs energy based on distance
- Use energy for: farming zones, questing, traveling to town, event participation

### Energy Optimization
```
✅ Plan multi-task trips: combine farming + quest objectives + town trading in one run
✅ Stack active Track tasks with daily quests so every action counts twice
✅ Watch ads for free energy restoration
✅ VIP/Super VIP daily bonuses restore significant energy
❌ Don't run back and forth for single tasks
❌ Don't waste energy on backtracking — plot efficient routes
```

### Energy Recovery Sources
| Source | Amount |
|--------|--------|
| Natural regen | ~1 per few minutes |
| Sleep (offline) | Fills slowly |
| Silver Coins | Instant full restore (costs coins) |
| Ad watch | Partial restore (free) |
| VIP daily bonus | Significant flat restore |
| Food items | Small partial restore |

## 8. Combat System

### Combat Basics
- **Hit and Run**: Attack, step back, repeat. Works on all enemy types
- Separate group enemies — pull small ones first, then fight the boss
- Use auto-attack for standard mobs; manual control for bosses and events

### Enemy Types & Counters
| Enemy Type | Best Weapon | Notes |
|------------|-------------|-------|
| Wild Animals | Rifles / Spears | Spirit skill reduces animal aggro |
| Bandits | Pistols / Knives | Headshot focus for fast kills |
| Boss Bandits | Any T3+ weapon | Hit-and-run mandatory |
| Piasa (Ash Grove) | T4–5 gear required | Prepare healing items |

### Gear Management in Combat
- Repair gear **before** it breaks — once at 0 durability it's gone
- Carry **lower-tier weapons** for weak mobs; save top gear for bosses
- Break down looted gear at ranch if it's obsolete — recover partial materials
- Always carry **food and healing supplies** — at minimum, grilled meat or cactus fruit

### Shootout Spots (World Events)
- Marked on the map; spawn periodically
- Contains good loot but stronger enemy groups
- **Check your gear level before engaging** — under-geared players will waste healing

## 9. Pet System (Wrangler's Track)

### Setup Requirements
1. Build **Pet House** at ranch
2. Build **Bait Workbench** (Production tab in Build Mode)
3. Unlock taming baits via **Wrangler's Skill Tree**

### Taming Process (Step-by-Step)
```
1. Craft appropriate bait for target animal
2. Travel to animal's spawn zone
3. Approach cautiously (use Stealth if no Spirit unlocked)
4. Place bait in quick-action slot → use it near animal
5. Timer starts (5 minutes) — immediately use Global Map to return to Ranch
6. Arrive at Ranch before timer expires
7. Keep Feeder stocked → animal becomes loyal companion
```

### Companion Benefits
- High-Spirit tamed animals will **fight alongside you**, reducing weapon use
- Reduces gear durability loss during zone farming
- Higher-tier pets deal significant damage and can tank hits

### Pet Breeding
| Requirement | Details |
|-------------|---------|
| Pet House Level | Must be **Lv.2+** to unlock Breeding tab |
| Animals needed | 2 of same species, **different genders** |
| Gender reveal | Unknown until animal is at ranch |
| Perk inheritance | T1–3: 1 perk inherited; T4–5: combo/new perks |
| Tier upgrade chance | Higher-tier parents → chance of upgraded offspring |
| Infertility | T3+ animals may be infertile → use **Fertility Elixir** |
| Growth | Wait in Pet House until adulthood to deploy |

### Fishing System (Wrangler — Lv.70+ Required)
```
1. Reach Level 70
2. Unlock Fishing in the Wrangler branch (costs Books + Silver)
3. Craft a Fishing Rod at Tool Bench
4. Travel to T4+ water zones (Shallow Lake, Marshlands)
5. Equip rod → interact with water body → fish
```
- **Buffalo Fish** (rare drop): when cooked → used to **repair gear** (extremely valuable)
- Regular fish → cooking recipes, pet food

## 12. Raiding & Alliance Systems

### Bandit Raids (Ranch Defense)
```
Anger Meter fills as you kill bandits in the world
→ When full: raid warning + countdown
→ Be ONLINE to defend manually (AI defense is weak)
→ Defeat the raid leader → loot Bandit's Stash Map
→ Use map to raid a bandit hideout for bonus resources
```

### Alliance (Guild) System
- Join early — alliance buildings provide persistent bonuses
- Alliance provides: specialized resources, group bonuses, Gold Rush access
- Active in alliance = access to **Canyon Trials** and shared tech

### Gold Rush (Alliance vs. Alliance Event)
```
1. Alliance pools Silver to buy a "Scouting Map" from Slick Johnny
2. Map reveals enemy alliance base location
3. 2-hour window opens for individual members to conduct raids
4. Objective: steal the enemy's Safe → move it to the map boundary
5. Progress from multiple raiders stacks (team effort)
6. Rewards: alliance resources + personal loot based on league tier
```

### Battle Pass / Season Pass
- Periodic seasonal events with tiered milestone rewards
- Currency: **Golden Bullets** (from daily + seasonal tasks)
- Progress → unlock: high-tier gear, blueprints, rare ingredients, ranch decorations
- **Premium Pass** option unlocks additional exclusive reward tiers throughout season
- **Key advice**: complete ALL daily tasks every day during active season pass

## 13. Patterns & Best Practices

### ✅ Do
- Log in **daily** — daily rewards compound into massive resource advantages over weeks
- Always **combine tasks** in one trip: quest objectives + farming + trading in one route
- **Repair before break** — never let weapons/armor hit 0 durability
- **Upgrade benches first**, gear second — infrastructure unlocks higher-tier crafting
- **Save healing items** (potions/ointments) for events/bosses; use basic food for daily play
- Store emergency food/supplies in **Inbox** — safe from raids, accessible anywhere
- **Watch all available ads daily** — meaningful free resource/energy gains
- **Stack Tracks** — always have 5 active simultaneously to maximize Book income
- Join an **active Alliance** as soon as possible — even casual alliances multiply rewards

### ❌ Don't
- **Don't rush leveling** — XP outpacing your gear = enemies too strong, base too weak
- **Don't let energy cap** at 100 — any regeneration above 100 is permanently lost
- **Don't enter high-tier zones under-geared** — gear loss instantly sets you back
- **Don't craft T1/T2 armor early** — use found gear + repair instead; save mats for bench upgrades
- **Don't use oats for anything except horse/wagon**
- **Don't ignore the Anger Meter** — let it fill without preparing to defend = resource theft
- **Don't reset Skill Tree nodes impulsively** — only 3 free resets per branch

## 15. Advanced Topics

### Late-Game Zone Rotation (Lv.75+)
```
Daily Route Example:
1. Ranch → collect production (Field, Well, Chicken Coop)
2. Pick up daily quests → identify overlapping objectives
3. Ash Grove run (T4–5 resources + bandit kills for season quest)
4. Return to Silverton → trade surplus, grab blueprint reset
5. Ranch → craft, repair, upgrade benches
6. Check Alliance chat → participate in Gold Rush if scheduled
```

### Efficient Blueprint Acquisition
1. Trade common tubes to Gustav (Silverton) for uncommon ones daily
2. Farm Old Mine (T3) for Skill Books used in Headhunter track
3. During events: prioritize blueprint rewards over decorations
4. Use Smuggler Adam McCoy when Rare blueprints are in his rotation

### F2P Survival (No-Spend Strategy)
- Never spend Gold on energy restores — Silver is sufficient
- Complete every daily task + event — these generate enough Silver to progress
- Focus Season Pass on free tier progressing — you don't need Premium to progress
- Watch every available daily ad — adds up to significant resource over time
- Trade resources in Silverton daily — this alone can bridge gear gap without crafting

### High-Level PvP Clarification
> Real-time open-world PvP does NOT exist in Westland Survival. "Enemy players" in the
> wild are AI-controlled copies. Real player interaction is:
> - Alliance Gold Rush raids (attacker is real, defender is AI during raid)
> - Seasonal leaderboards and Canyon Trials rankings
